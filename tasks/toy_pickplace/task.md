# Toy Pickplace

This placeholder task exists to smoke test the Docker benchmark harness.

Your final submission must be written to:

```text
/workspace/output/final_submission/result.json
```

The placeholder evaluator expects JSON with a numeric score:

```json
{"score": 0.2}
```

Scores are clamped to the range `[0, 1]` and reported as `success_rate`.
